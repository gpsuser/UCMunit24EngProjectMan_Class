**Question 1** *(Source: January 2023, Question 4(a)(i))* **[4 marks]**

Better Health plc is undertaking a health monitoring technology project. Johnny is the project manager. During implementation, Johnny reviews the project issue log. The table below sets out four severity levels and their definitions.

```
+----------------+--------------------------------------------------------------+
| Severity Level | Definition                                                   |
+----------------+--------------------------------------------------------------+
| Critical       | The project cannot continue without immediate management     |
|                | action and escalation to senior stakeholders.               |
+----------------+--------------------------------------------------------------+
| High           | The project is significantly affected and may not meet key  |
|                | objectives without urgent action being taken.               |
+----------------+--------------------------------------------------------------+
| Medium         | The project is affected but can continue. Action is         |
|                | required within an agreed timeframe.                        |
+----------------+--------------------------------------------------------------+
| Low            | Minor impact on the project. Can be managed as part of      |
|                | normal working.                                             |
+----------------+--------------------------------------------------------------+
```

Using the severity definitions above, classify each of the following issues recorded in Johnny's issue log.

```
+----+------------------------------------------------------------------+--------------------+
| #  | Issue                                                            | Your Classification|
+----+------------------------------------------------------------------+--------------------+
| A  | A key supplier has delivered faulty components, halting all      |                    |
|    | production activities until replacements can be sourced.         |                    |
+----+------------------------------------------------------------------+--------------------+
| B  | A team member will be absent for two days due to a minor illness.|                    |
+----+------------------------------------------------------------------+--------------------+
| C  | A subcontractor has reported a three-week delay on an activity   |                    |
|    | that does not sit on the critical path.                          |                    |
+----+------------------------------------------------------------------+--------------------+
| D  | The approved project budget has been fully spent, with 20% of    |                    |
|    | work still to be completed and no contingency remaining.         |                    |
+----+------------------------------------------------------------------+--------------------+
```

*(1 mark for each correct classification)*

**Question 3** *(Adapted from: January 2023, Question 4(a)(iii))* **[3 marks]**

StructBuild Ltd is a structural engineering firm constructing a new pedestrian bridge in Cardiff. The project manager, Priya, is monitoring progress and has noted that the project is running two weeks behind schedule. Her monitoring report identifies that three specialist welders employed by a subcontractor have been reassigned to a higher-priority contract elsewhere, leaving no qualified welding resource available on Priya's site.

Explain why the loss of these specialist resources could lead to **time creep** on Priya's project.

**Question 4** *(Adapted from: June 2024, Question 4(b))* **[5 marks]**

TechForge Ltd manufactures precision aerospace components. During the implementation stage of a new production line, the quality manager carries out quality control sampling on a batch of 60 titanium brackets. The specified tensile strength is 420 MPa, with a tolerance of ± 15 MPa, giving the following control limits:

```
  Upper Control Limit (UCL) = 435 MPa
  Lower Control Limit (LCL) = 405 MPa
```

The results of the quality control sample are shown in the table below.

```
+------------------------+--------------+-------------------------------+
| Tensile Strength (MPa) | Frequency (f)| Strength x Frequency (MPa)    |
+------------------------+--------------+-------------------------------+
| 405                    | 2            | 810                           |
| 410                    | 10           | 4,100                         |
| 415                    | 20           | 8,300                         |
| 420                    | 18           | 7,560                         |
| 425                    | 8            | 3,400                         |
| 430                    | 2            | 860                           |
+------------------------+--------------+-------------------------------+
| TOTAL                  | 60           |                               |
+------------------------+--------------+-------------------------------+
```

**(a)** State the quality characteristic being tested in this quality control check.
**[1 mark]**

**(b)** Calculate the mean tensile strength of the sample. Show your working clearly.
**[2 marks]**

**(c)** State whether the manufacturing batch passes or fails the quality control check. Justify your answer by referring to the control limits.
**[2 marks]**

**Question 5** **[4 marks]**

RailTech Engineering is managing the installation of new signalling equipment along a 40-kilometre section of railway. At the end of Week 6 of the implementation stage, the project monitoring team records the following Earned Value Analysis (EVA) data:

```
+---------------------------+-----------+
| Metric                    | Value     |
+---------------------------+-----------+
| Planned Value      (PV)   | £240,000  |
| Earned Value       (EV)   | £192,000  |
| Actual Cost        (AC)   | £210,000  |
+---------------------------+-----------+
```

**(a)** Calculate the **Schedule Variance (SV)** for the project at the end of Week 6. State whether this is favourable or unfavourable.
**[1 mark]**

**(b)** Calculate the **Cost Performance Index (CPI)** for the project at the end of Week 6. Give your answer to two decimal places.
**[1 mark]**

**(c)** Explain what your CPI result from part (b) tells the project manager about the financial performance of RailTech's signalling project.
**[2 marks]**

**Question 6** **[2 marks]**

A project manager at a civil engineering firm is using a **milestone slip chart** to monitor progress on a bridge construction project. The chart shows that the first three milestones were achieved on time. However, the fourth milestone — completion of the structural frame — was reached two weeks later than planned.

Explain **one** reason why a milestone slip chart is a useful control technique for this project manager in this situation.

**Question 7** **[3 marks]**

Explain the difference between **quality assurance (QA)** and **quality control (QC)** in the context of monitoring an engineering project. In your answer:

- State whether QA is proactive or reactive, and give **one** example of a QA activity.
- State whether QC is proactive or reactive, and give **one** example of a QC activity.
