## A Note on Exam Alignment for Week 22

The *All Exam Alignment Table* confirms that Week 22 — Assessment Review & Remediation — has **no examination questions directly mapped to it** across any past paper series (January 2022 to June 2024). This is by design: Week 22 is a consolidation and remediation session rather than a content-delivery week, and its purpose is to correct errors identified in the mid-term assessment across LO1–LO4.

The two questions in **Section A** have therefore been selected from pre-2025 examination papers on the basis that they directly test the learning outcomes which the Week 22 lecture explicitly identifies as the areas most frequently associated with student error. Both are reproduced verbatim and the source paper and question reference are stated clearly for each.

### Question 1

**Source: Question paper January 2024 (Unit 24), Question 1(b)**
**Learning Outcome: LO1.2 — Project planning: outline plan**

---

Read the following case study.

> HD Tyre Ltd is a UK manufacturer that produces tyres for a wide range of vehicles. The company has decided to implement a project to reduce the carbon footprint of its manufacturing processes. The project will involve changes to machinery, supplier contracts, and energy sources across two production sites in the Midlands.

### Question 2

**Source: Question paper January 2024 (Unit 24), Question 1(c)**
**Learning Outcome: LO4.4 — How to judge the validity of information**

---

HD Tyre Ltd's project manager has been collecting information from a range of sources to support decisions during the project.

**(c)** Draw a line to match each **validity criterion** in the left-hand column to the correct **definition** in the right-hand column.

```
+---------------------+        +------------------------------------------------+
|                     |        | The information can be reproduced — if the     |
|     Quality         |        | research were repeated, the same results would  |
|                     |        | be obtained.                                   |
+---------------------+        +------------------------------------------------+
                               | The information is relevant to the needs of the |
|     Reliability     |        | project and has sufficient detail and accuracy  |
|                     |        | to meet the required standard.                 |
+---------------------+        +------------------------------------------------+
                               | The information is up to date and still         |
|     Importance      |        | applicable to the current situation.            |
|                     |        |                                                |
+---------------------+        +------------------------------------------------+
                               | The information is significant to the business  |
                               | and would affect project decisions if it were   |
                               | found to be incorrect or unavailable.           |
                               +------------------------------------------------+
                               | The information comes from a credible and       |
                               | trustworthy source with no reason for           |
                               | distortion.                                     |
                               +------------------------------------------------+
```

*(3 marks)*

### The Apex Structures Scenario

Read the following case study. It applies to both Question 3 and Question 4.

> **Apex Structures Ltd** is a civil engineering firm managing the construction of a new maintenance depot for a light rail operator in the West Midlands. The depot will include covered storage for twelve rail vehicles, a workshop, offices, and a specialist electrical substation. The project has a budget of £3.5 million and a contractual deadline of 16 months. Failure to complete within the deadline will result in financial penalties payable to the rail operator.

---

### Question 3

**Adapted from: Question paper January 2024 (Unit 24), Question 1(b)**
**Learning Outcome: LO1.2 — Project planning: outline plan**

---

Apex Structures Ltd's project manager, Ravinder, has just entered the planning stage of the depot project.

**(a)** Identify and describe **two** topics that should be included in the outline plan for this project. For each topic, explain why it is particularly important given the specific context of the Apex Structures depot project.

*(6 marks — 3 marks per topic: 1 mark for identification and description; 2 marks for contextualised explanation)*

---

### Question 4

**Adapted from: Question paper January 2024 (Unit 24), Question 1(c)**
**Learning Outcome: LO4.4 — How to judge the validity of information**

---

Ravinder has gathered three research reports from different sources to inform project decisions. He must evaluate the validity of each report before using its findings.

**(b)** Draw a line to match each **validity criterion** in the left-hand column to the correct **definition** in the right-hand column.

```
+---------------------+        +------------------------------------------------+
|                     |        | The information is free from errors and         |
|     Accuracy        |        | correctly represents the facts as they          |
|                     |        | actually are.                                  |
+---------------------+        +------------------------------------------------+
                               | The information has been influenced by          |
|     Currency        |        | personal opinion or a vested interest, which   |
|                     |        | may distort the findings.                      |
+---------------------+        +------------------------------------------------+
                               | The information is up to date and remains       |
|     Bias            |        | applicable to the current project context.      |
|                     |        |                                                |
+---------------------+        +------------------------------------------------+
                               | The information is significant to the           |
                               | business and would affect decisions if          |
                               | it were unavailable.                           |
                               +------------------------------------------------+

                               | the research process would yield the same      |
                               | results.                                       |

```

### Question 5

**Learning Outcome: LO2.2 — Skills needed: leadership styles**

---

Ravinder is responsible for a team of ten engineers and technicians. The team includes four newly qualified engineers who have limited site experience and require close guidance, and six senior engineers who are highly skilled, self-motivated, and capable of working independently.

**(a)** Identify and explain **two** leadership styles that Ravinder could adopt for different members of his team. For each style, explain why it would be appropriate for the specific group of team members described.

*(6 marks — 3 marks per style: 1 mark for correctly identifying the style; 2 marks for a contextualised explanation linked to the team members described)*

---

### Question 6

**Learning Outcome: LO3.3 — PERT technique**

---

The Apex Structures project manager has asked the structural engineering team to estimate the duration of the steel frame erection activity for the depot building. The team provides the following time estimates:

```
+------------------------------+----------+
| Estimate Type                | Duration |
+------------------------------+----------+
| Optimistic time (O)          |  5 weeks |
| Most likely time (M)         |  8 weeks |
| Pessimistic time (P)         | 17 weeks |
+------------------------------+----------+
```

**(a)** Using the PERT formula, calculate the **expected duration (E)** for the steel frame erection activity. Show your working clearly.

*(3 marks — 1 mark for correct formula; 1 mark for correct substitution; 1 mark for correct answer)*

---

### Question 7

**Learning Outcome: LO4.5 — How to collate and use information**

---

The project manager has collected the following data on weekly concrete pour volumes during the foundation stage of the depot project, compared with the planned target.

```
+------+--------------+-------------+---------------+
| Week | Planned (m3) | Actual (m3) | Variance (m3) |
+------+--------------+-------------+---------------+
|  1   |     120      |     115     |      -5       |
|  2   |     120      |     108     |     -12       |
|  3   |     120      |     122     |      +2       |
|  4   |     120      |     110     |     -10       |
+------+--------------+-------------+---------------+
|Total |     480      |     455     |     -25       |
+------+--------------+-------------+---------------+
```

The project requires a minimum average weekly pour volume of **115 m³** to remain on schedule.

**(a)** Using the data in the table, evaluate whether the project is meeting its scheduled concrete pour target. Justify your conclusion with specific reference to values from the data above.

*(4 marks — 1 mark for a clear decision; 1 mark for correct calculation of the mean; 2 marks for a well-justified evaluation that refers to specific data values and identifies a trend or pattern)*
